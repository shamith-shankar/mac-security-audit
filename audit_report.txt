----------------------------------------
What: Is your hard drive encrypted?
Why: Encryption scrambles your data so no one can read it if they steal your Mac.
Without it, anyone could get your files, photos, and passwords.

CHECK 3: SYSTEM INTEGRITY PROTECTION (SIP)
----------------------------------------
What: Is your system protected from file changes?
Why: SIP stops anyone, even with admin access, from messing with important
system files. This blocks malware and other bad stuff from breaking your Mac.

✅ PASS — System Integrity Protection (SIP) is enabled


CHECK 4: AUTOMATIC SECURITY UPDATES
----------------------------------------
What: Does your Mac automatically get security updates?
Why: Hackers find holes in software all the time. Updates fix these holes so
your Mac stays safe. Automatic updates make sure you never miss one.

❌ FAIL — Automatic security updates are disabled
Fix: Turn on automatic updates in System Preferences → General → Software Update


CHECK 5: GATEKEEPER & XPROTECT (MALWARE PROTECTION)
----------------------------------------
What: Does your Mac check if apps are safe before you run them?
Why: Gatekeeper blocks suspicious apps and malware from running on your
computer. It keeps track of apps that have viruses or sketchy code.

❌ FAIL — Gatekeeper is disabled (assessments disabled)
Fix: Enable in System Preferences → Security & Privacy or Terminal: spctl --master-enable


CHECK 6: GUEST ACCOUNT STATUS
----------------------------------------
What: Can guests log into your Mac without a password?
Why: Guest accounts are for when friends or family want to use your Mac.
If a guest account is on, anyone who walks up can access your files and data.
